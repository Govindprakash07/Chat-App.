# Chat App 💬  

A simple real-time chat application using Node.js, Express, and Socket.io.  

## Features  
✔️ Real-time chat  
✔️ Simple and clean user interface  
✔️ No need for page reloads  

## Technologies Used  
- Node.js  
- Express.js  
- Socket.io  
- EJS for templating  

## How to Run  
1. Install **Node.js**.  
2. Clone the repo:  
   ```sh
   git clone https://github.com/YOUR_USERNAME/Chat-App.git
   cd Chat-App
